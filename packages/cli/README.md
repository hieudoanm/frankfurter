## 📦 Installation

```bash
curl -fsSL https://github.com/hieudoanm/frankfurter/raw/refs/heads/master/packages/cli/bin/frankfurter -o frankfurter
chmod +x frankfurter
sudo mv frankfurter /usr/local/bin/
```

## 🚀 Usage

```bash
frankfurter --from EUR --to USD --amount 1
```
